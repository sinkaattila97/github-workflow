# GitHub Workflow Tanulási Projekt

Ez a projekt egy egyszerű Java alkalmazás, amely bemutatja a GitHub workflow-t, pull requesteket és code review-kat.

## Projekt Leírása

A projekt egy egyszerű könyvtár kezelő alkalmazás, amely könyveket tárol és alapvető műveleteket végez velük (hozzáadás, listázás, keresés).

## Fájlstruktúra

```
src/
├── main/
│   └── java/
│       └── com/
│           └── example/
│               └── library/
│                   ├── Book.java          # Könyv modell osztály
│                   ├── Library.java       # Könyvtár kezelő osztály
│                   └── LibraryApp.java    # Főprogram
└── test/
    └── java/
        └── com/
            └── example/
                └── library/
                    ├── BookTest.java      # Book osztály tesztjei
                    └── LibraryTest.java   # Library osztály tesztjei
```
